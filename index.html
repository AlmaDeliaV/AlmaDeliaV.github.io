Conceptos Basicos de Administracion de Bases de Datos

¿Qué son los Datos?

Un dato es una unida de información, ya sea cualitativa o cuantitativa; la unión de diferentes datos genera un mensaje o información, esta información puede tener distintos usos. Por ejemplo las transacciones de una empresa, el pago a los empleados, si unimos todos estos datos obtendremos información importante y vital para la empresa.

En nuestro día a día ocupamos y generamos gran cantidad de datos, desde el registro a una nueva plataforma de internet, el hacer compra, entre otras actividades que generan datos, así como utilizar aquellos que ya están almacenados, por ejemplo el buscar en un navegador como; Google, mega, etc.

¿Para qué sirven los datos?

Los datos nos brindan información y conocimiento de cierta situación que nos interese. Pero no solo eso los datos también nos permiten tomar decisiones de forma objetiva, al contar con información que respalde esas decisiones en base a estadísticas y datos científicos.

Por ejemplo, que colores son mas atractivos para diseñar una pagina web en base al tema que se exponga en dicha página.

Según el análisis los colores más recomendables para una página web sobre comida rápida el color que debes incluir si oso es el rojo y el amarillo ya que el cerebro los asocia con la comida. Un claro ejemplo es McDonald’s, Telepizza o Burger King.

¿Cuál es la importancia de los datos en la empresa?

Sin importar el tipo de empresa que se tenga el uso de datos es muy importante ya que la unión de eso datos genera información coherente de utilidad para la empresa. Datos de compras, ventas, proveedores, clientes, etc. Permiten a la empresa llevar un control de procesos. Esto permite que una empresa conozca el estado de los distintos departamentos, el tener una base con el historial de los datos permite hacer comparativas para conocer el desarrollo y crecimiento de la empresa

¿Es importante la seguridad en los datos?

La seguridad de los datos es muy importante, porque muchas veces esos datos pueden ser de gran importancia como las contraseñas, números de cuenta, entre otros.

La importancia de la ciberseguridad radica en que muchas vece la información de la red no es segura, y la vulnerabilidad a ataques virtuales es cada vez mayor, así como la infección de malware

Es por eso por lo que la ciberseguridad busca crear formas de proteger esos datos para evitar robos o perdidas de información importante. Se quiere proteger a los documentos y sistemas dentro de una empresa para evitar virus, robo y/o extorsiones.

Este tipo de seguridad mantiene en confidencialidad los documentos, contratos, datos de personal, así como los procesos que lleva a cabo una compañía para aumentar su productividad de forma eficaz.

¿Por qué los datos requieren un formato definido?

Se entiende por formato a “la estructura interna y codificación de un objeto digital, que permite que éste sea procesado o presentado en una forma accesible”. existen diferentes formatos según el tipo de datos que se trate. Así, por ejemplo, hay formatos comunes para texto, imagen o sonido, y otros que son frecuentemente utilizados sólo dentro de ciertas disciplinas o comunidades de investigación.

Cuando almacenamos datos dentro de una tabla de BD debemos asegurarnos que la información sea del mismo tipo de dato, además de que este dentro del dominio de ese atributo.

La razón por la que los datos deben seguir un formato definido, es para evitar ambigüedades de la información, para que los datos puedan ser gestionados y almacenados de manera adecuada, por ejemplo no guardar una cadena de texto en lugar de un numero telefónico, y como este hay mucho ejemplos más. Si todos los datos siguen un formato especifico el almacenamiento en tablas para guardarlos en la base de datos será fácil y ágil, así como el extraerlos y usarlos.

¿Qué papel juega el ABD en los datos de la empresa?

Un ABD (administrador de base de datos o DBA) es un profesional responsable de la gestión de la información dentro de una empresa. Sus funciones principales son:

  • Asegurar el correcto funcionamiento de la base de datos
  • Proteger y retener la información de la BD
  • Evitar pérdidas de datos
  • Solucionar incidencias en los datos

Es decir, un DBA es el encargado de que toda la información de la empresa este guardada y respaldada para cualquier consulta
